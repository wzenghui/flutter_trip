## 如何运行？

1. 安装和配置Flutter开发环境(如已经配置过可跳过)，可参考课程[Flutter入门：开发工具准备与开发环境搭建](https://coding.imooc.com/class/321.html)一章的讲解。
2. 在项目根目录运行`flutter run`或用装有Flutter和Dart插件的IDE运行；
